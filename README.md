# 🧠 Psychology Appointment System

A web-based appointment management system for psychologists and clients built with ASP.NET Core MVC.

---

## 📌 Overview

This system helps psychologists manage appointments and clients book available time slots easily.

It includes:
- Counselor management
- Client registration
- Appointment booking system
- Weekly schedule calendar
- Automatic prevention of double booking

---

## ⚙️ Technologies Used

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Razor Views
- jQuery

---

## 🧩 Features

### 👨‍⚕️ Counselors
- Add / edit / delete counselors
- View counselor dashboard

### 👤 Clients
- Register clients
- Store personal information

### 📅 Appointments
- Select counselor
- Choose date
- View available time slots
- Book appointment
- Prevent duplicate bookings

### 📊 Weekly Calendar
- Weekly view per counselor
- Shows booked and free slots
- Displays client names

---

## 🗄️ Database

- SQL Server
- Entity Framework Core (Code First)

Tables:
- Counselors
- Clients
- Appointments

---

## 🚀 How to Run

1. Clone repository:
git clone https://github.com/USERNAME/PsychologyAppointmentSystem.git
